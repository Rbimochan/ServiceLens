# Phase 14 - Interaction Profile Discovery

## RQ3

**What support interaction profiles emerge through unsupervised learning?**

The selected clustering solution identifies 2 empirical profiles: Efficient Resolution Interactions, Dissatisfied High-Effort Interactions.

### Cluster 0: Efficient Resolution Interactions

- Size: 69,464 (83.92%)
- Average CSAT: 4.806
- Average response time: 135.2 minutes
- Dominant channel: Inbound
- Dominant category: Returns
- Dominant sub-category: Reverse Pickup Enquiry
- Dominant tenure: >90
- Dominant shift: Morning
### Cluster 1: Dissatisfied High-Effort Interactions

- Size: 13,315 (16.08%)
- Average CSAT: 1.265
- Average response time: 389.5 minutes
- Dominant channel: Inbound
- Dominant category: Returns
- Dominant sub-category: Reverse Pickup Enquiry
- Dominant tenure: >90
- Dominant shift: Morning

The anticipated efficient and high-effort patterns are retained only where supported by measured response-time and CSAT statistics. Remaining labels reflect each cluster's actual dominant operational characteristics.
