# Phase 10 - Dataset Overview

## Project Objective

ServiceLens aims to investigate drivers of customer satisfaction and discover customer-support interaction patterns using scalable analytics with Python, PySpark, and Tableau.

## Dataset Source

- Source: Kaggle Customer Support Ticket Dataset
- URL: https://www.kaggle.com/datasets/akashbommidi/customer-support-data
- Local raw file: `data/raw/customer_support_tickets.csv`
- Prepared file: `data/processed/customer_support_tickets_prepared.csv`

## Verified Dataset Profile

| Item | Verified value |
|---|---|
| Prepared rows | 85,907 |
| Prepared columns | 52 |
| Original retained columns | 20 |
| Engineered columns | 32 |
| Issue-report date range | July 28, 2023 20:42 to August 31, 2023 23:58 |
| Response date range | August 1–31, 2023 |
| Survey date range | August 1–31, 2023 |
| Primary target | `CSAT Score` |

`CSAT Score` is measured on a 1–5 scale and is complete for all records.
