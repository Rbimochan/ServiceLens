# ServiceLens

ServiceLens is a customer satisfaction analytics project focused on understanding support ticket data and preparing future analysis with Python, Pandas, PySpark, Jupyter, and Tableau.

## Project Aim

The project aims to analyze customer support interactions and prepare a structured workflow for customer satisfaction analytics, including future exploratory analysis, modeling, and dashboarding.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- PySpark
- Jupyter
- Tableau
